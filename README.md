# URL Shortener

Next.JS ile hazırlanmış basit ve hızlı URL kısaltma servisi.

## Kurulum

    $ npm install
    # or
    $ yarn install

## Düzenleme

- .env.example dosyasını .env olarak değiştirip ilgili kısımları düzenleyin
- public/manifest.json

## Çalıştırma

### Geliştirme Ortamı

    $ npm run dev
    # or
    $ yarn dev

### Üretim Ortamı

#### Oluştur

    $ npm run build
    # or
    $ yarn build

#### Başlat

    $ npm run start
    # or
    $ yarn start
